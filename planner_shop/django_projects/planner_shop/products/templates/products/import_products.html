Требования к JSON файлу:
  • Обязательные поля: title, product_type, price, sku
  • Допустимые типы товаров: planner, sticker, kit, accessory
  • Цена: должна быть положительным числом
  • Артикул (SKU): должен быть уникальным
Пример валидного JSON:
{
    "title": "Набор стикеров 'Флора'",
    "product_type": "sticker", 
    "description": "Набор виниловых стикеров с цветочными мотивами",
    "price": "299.00",
    "sku": "STK001",
    "is_available": true,
    "stock_quantity": 100
}
